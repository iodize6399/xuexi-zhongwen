# Part of Day

acceptable tolerance of **±30 minutes**

| Part of Day | English         | Time Range          |
| :---------- | :-------------- | :------------------ |
| 早上          | Early Morning   | Sunrise ~ 8:00 AM   |
| 上午          | Late Morning    | 8:01 AM ~ 11:59 AM  |
| 中午          | Noon / Midday   | 12:00 PM ~ 12:30 PM |
| 下午          | Afternoon       | 12:31 PM ~ 5:00 PM  |
| 晚上          | Evening / Night | 5:01 PM ~ 11:59 PM  |
# Expressing time

Past the hour: `[Hour] + 点 + [Minutes] + 分`

To/Before the Hour: `差 (chà) + [Minutes Left] + 分 + [Next Hour] + 点(钟)`

| Time     | Chinese         | English            |
| :------- | :-------------- | :----------------- |
| **6:00** | 六点钟             | Six o'clock        |
|          | 六点              | Six o'clock        |
| **6:15** | 六点十五分           | Six fifteen        |
|          | 六点一刻            | A quarter past six |
| **6:30** | 六点三十分           | Six thirty         |
|          | 六点半             | Half past six      |
| **6:45** | 六点四十五分          | Six forty-five     |
|          | 六点三刻            | Six forty-five     |
|          | ==差十五分七点钟==<br> | Fifteen to seven   |
|          | ==差一刻七点钟==      | A quarter to seven |

# Asking the current time

[[Question Words#几]]

## To say the current time along with part of day
`现在 + 是 + Part of Day + Time Phrase`

>现在==是==下午两点三刻。
>Currently, it's 2:45 in the afternoon.

To answer only the current time:
`现在 + Time Phrase`

> 现在两点三刻。

# Negating time phrases

>现在==不是==八点钟。
>The time is not 8'o clock now.
