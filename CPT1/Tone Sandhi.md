# 一

`yi1 and 1/2/3` changes to `yi4 and 1/2/3`

`yi1 and 4/5`[^1] changes to `yi2 and 4/5`

# Changes in 3rd tone

`3 by 1/2/3` changes to `half 3rd tone (sans rising terminal)`

`3 by 3` changes into `2 and 3`

>nǐhǎo - níhǎo

# 不

bu is 4th tone by default

`bu4 and 4` changes to `2 and 4`
>bu2ke4qi 不客气

`bu4 and 1/2/3` will remain as `4 and 1/2/3`
>bu4mang2 不忙



[^1]: 5th tone is sometimes called the neutral tone
