# Type 1

Read 是 as neutral tone

>他是学生，他不是老师。
>He is a student, he is not a teacher.

# Type 2

Gives more emphasis than the first.

>这件毛衣==是==谁==的==？
>Whose sweater is this?
>Same as saying: 谁==的==毛衣？


>今天的电影==是==中文==的==。
>Today's movie is Chinese.
>Same as saying: 今天==的==中文电影

# Negating

For both type 1 and type 2, put `不` before `是`

>那==不是==画报，那是本子。
>That is not a pictorial, that is a notebook.

>这本小说==不是==老师==的==。
>This is not the teacher's novel.
>那件毛衣==不是==他==的==。
>This is not his sweater.
