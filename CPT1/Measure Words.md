- numbers cannot modify a noun directly
- must accompany a measure word in between
- every noun has a specific measure word

# 口

- For family members, wells

# 张

- For flat surfaces
- Tables, map, beds, ticket, ==picture==, newspaper, paper
# 本

- For things having bindings
- Books, novels, magazines, dictionary, ==pictorial==
# 支

- For barrel like objects
- pens, pencils
# 把

- For tables
# 位

- For honorable persons
- Doctor, teachers, etc.
# 篇

- For articles
# 个

- General measure word, for anything not covered by above
 
>丁文有一个外国朋友。
>Ding Wen has a foreign friend.

# 些

See: [[Usage of 些]]

# 节

- For lessons

>我一个星期==只上五节汉语课==。
>I attend only 5 Chinese classes in a week.

# 只 [^1]

- For animals having legs or wings
- Dogs, cats

# 间 [^1]

- For rooms

---

# Use 两 for numeral 2

When denoting numeral 2 before a measure word, use `两` NOT `二`

>我在操场看见两位老师。
> I saw two teachers at the playground.
