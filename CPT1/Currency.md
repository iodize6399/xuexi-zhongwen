# Units of currency

块：==k==uài (元：==y==uán）
毛：==m==áo (角： ==j==iao）
分：==f==ēn

Either use `k-m-f` or `y-j-f`

>[!tip]
>When only one unit is to be said, always suffix `钱`
>>45.00 元：四十五块钱

# Numeration of numbers

For number from 110 to 119: always prefix 一 to 十

>一百==一==十九
>119

# Using Zero 零

Mark the unit with `零` as zero and express the final unit
17.04元：十七块零四分

# Noteworthy sentences

>你==还要==别的吗？
>Do you still want anything else?
>我==想要==那一件毛衣。
>I would like to have that sweater.

>您要什么？
>What do you want?

>...一共多少钱？
>How much in total?

>颜色的很好，不大，==也不==小。
>Color is very nice, is ==neither== big ==nor== small.
