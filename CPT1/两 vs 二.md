# For currency

2 occurring in the beginning: use `两`

>2.1元：两块一毛
>0.25元：两毛五分
>0.02元：两分钱

 2 occurring in the middle: use `二`
 
> 1.25元：一块二毛五分

# Before a measure word

Before a [[Measure Words]] or a noun which does not need a measure word

>两支钢笔
>2 pens

# Before Month

For February, it is `二月`

# For Day

For Tuesday, it is `星期二`

# For time

Use `两`

> 两点五十五
> 2:55
